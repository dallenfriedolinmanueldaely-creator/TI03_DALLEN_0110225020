Membuat List dl

HTML
Adalah singkatan dari HyperText Markup Language, bahasa standar yang digunakan untuk membuat dan menyusun kerangka dasar halaman web. Fungsinya adalah untuk mendefinisikan struktur konten seperti judul, paragraf, gambar, dan tautan, yang kemudian ditampilkan oleh peramban web (browser).

CSS
Adalah singkatan dari Cascading Style Sheets, sebuah bahasa stylesheet yang digunakan untuk mengatur tampilan dan gaya dokumen yang ditulis dalam bahasa markup seperti HTML. Fungsinya adalah untuk memisahkan struktur konten (HTML) dari presentasi visual, sehingga memungkinkan pengembang untuk mengontrol tata letak, warna, font, dan elemen visual lainnya agar halaman web lebih menarik, konsisten, dan responsif.

JavaScript
Adalah bahasa pemrograman yang digunakan untuk membuat halaman web menjadi interaktif dan dinamis. Bersama dengan HTML dan CSS, JavaScript adalah salah satu dari tiga teknologi inti dari World Wide Web, yang bertanggung jawab untuk menambahkan fungsionalitas seperti animasi, pembaruan konten secara real-time, dan interaktivitas lainnya.
